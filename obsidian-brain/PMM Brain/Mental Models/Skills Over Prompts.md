# Skills Over Prompts

## Description
A prompt is a conversation you have once. A skill is a system that persists. Best AI marketers build persistent systems: encoded audience profiles, voice style cards, quality bars, workflow pipelines. Prompting re-explains context every time; a skill remembers forever. Three assets: (1) Audience profile from research. (2) Writing style card from best posts. (3) Workflow: Setup → Ideation → Enhancement.

## How to Apply
Build foundational assets stored in persistent memory. Separate workflows into sequential skills. Never re-explain what can be remembered.

## When to Use
When AI content sounds generic. When scaling content beyond manual prompting.
