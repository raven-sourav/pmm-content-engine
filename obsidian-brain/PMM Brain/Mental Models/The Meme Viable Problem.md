# The Meme Viable Problem

## Description
Problem validation: can you meme it? If shareable as meme: enough people experience it, emotionally resonant, creates community. Three steps: find frustration, meme it, measure resonance.

## How to Apply
Express the problem as a meme. If people share/tag friends, it's real. If flat, not visceral enough.

## When to Use
Problem validation. Testing if pain is widespread enough for a business.
