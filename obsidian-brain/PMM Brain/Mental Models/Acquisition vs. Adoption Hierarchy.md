# Acquisition vs. Adoption Hierarchy

## Description
Acquiring new customers costs 10x more than retaining. Upselling existing customers is 5-10x cheaper than acquisition. Seven-stage SaaS user journey: Aha Moment → Activated → Selected → Paid → Basic → Pro → Advocate. Focus investment on moving existing customers up the ladder.

## How to Apply
Measure adoption by source (which acquisition channels yield highest adoption). Track four adoption dimensions: Breadth, Depth, Time to Adopt, Duration. Segment analysis by persona.

## When to Use
When churn is high, when acquisition costs are rising, when subscription-based growth plateaus.
