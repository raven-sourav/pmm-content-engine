# Content Ecosystem Model

## Description
3-layer social ecosystem: Founder (authority → demand), Team (reach → amplification), Company page (product → conversion). Most startups activate only one layer.

## How to Apply
Map active vs dormant layers. Activate at least 2 of 3 before scaling any single one.

## When to Use
When planning B2B social. When founder content works but pipeline doesn't follow.
