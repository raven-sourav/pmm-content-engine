# K.P.O. Goals Framework

## Description
Marketing planning with three goal categories: KPIs (metrics to hit), Projects (big-bet initiatives that change trajectory), Ops (behind-the-scenes: hiring, tooling, research). 'If you don't set a goal for it, it won't get prioritized.' Most teams skip Ops and it gets sacrificed every time.

## How to Apply
Categorize all work into K, P, and O. Budget explicitly for Ops. Build bottom-up AND top-down forecasts that validate each other.

## When to Use
During annual/quarterly marketing planning.
