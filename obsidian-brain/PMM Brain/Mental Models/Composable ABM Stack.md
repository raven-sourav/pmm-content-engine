# Composable ABM Stack

## Description
Replace bundled ABM platforms ($32K+ with 2-year lock-in) with composable BYOK architecture. Direct API enrichment costs $0.06-0.10 vs platform credits at $0.17. At 10K monthly enrichments, saves $700-1,100/month. 60-70% lower ABM costs, data coverage jumps from 40% to 85%.

## How to Apply
Build with HubSpot + Clay for list building, LinkedIn Campaign Manager for ads. Track 5 account stages: Identified → Aware (50+ impressions) → Interested (5+ clicks) → Considering (demo) → Selecting (deal). Target $10 pipeline per dollar.

## When to Use
When ABM platform quotes exceed budget. When arguing for composable vs monolithic MarTech.
