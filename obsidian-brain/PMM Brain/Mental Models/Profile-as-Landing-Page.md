# Profile-as-Landing-Page

## Description
LinkedIn profile as conversion funnel: Headline = value prop, Banner = CTA, About = problem-solution for buyer, Featured = lead magnets, Experience = case studies.

## How to Apply
Audit each zone: does this make a buyer want to learn more? Replace resume language with buyer-facing.

## When to Use
Before any LinkedIn push. When followers aren't converting to pipeline.
